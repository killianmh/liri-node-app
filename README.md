# liri-node-app

This app runs in Git Bash. To call it, use "node liri.js" and then pick one of the following commands:

-my-tweets: gives the last 20 tweets of my account
-spotify-this-song: returns song information about the song entered
-movie-this: returns movie information about the movie entered
-do-what-it-says: reads from the random.txt file and executes the command in that file (this can be changed to any of the three commands above)
